Fichier Val_postions : 
Créer un fichier texte avec toutes les valeurs des 3 variables. Ce fichier doit, en première ligne, comporter les noms des variables en majuscules séparés par ":" tel que : "POS:LB:LH" pour 3 variables. Dans ce même fichier, les valeurs à utiliser doivent être en colonne en dessous des noms de variable et toujours séparés par ":" tel que : "0.03:0.06:0.6" pour 3 variables.

Fichier .geo : 
Ce fichier doit avoir une bonne syntaxe. Si la syntaxe est mauvaise, le programme vous renverra la ligne qui fait défaut. De plus, le nom des variables dans le fichier doit être en majuscule. Enfin, la commande 'extrude' est écrite par gmsh sur Trois ligne, pour que le programme marche il vous suffit d'enlever les retour a la ligne, en laissant les espaces
